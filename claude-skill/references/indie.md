# 퍼즐 / 인디 장르 창작 가이드

## 핵심 특성
인디/퍼즐은 **아이디어 하나를 끝까지 밀어붙이는 것**이 핵심. 규모보다 밀도.

## 세계관 설계 포인트
- 세계관 전체가 **하나의 은유 또는 테마**를 담고 있을 때 가장 강렬함
  - 예: 기억 상실 → 퍼즐 조각 맞추기 / 외로움 → 연결 메커닉
- 설명 없이 **시각·청각·인터랙션**으로 세계를 보여주는 설계 권장
- 작은 세계라도 **일관된 내부 논리** 필요

## 시나리오 설계 포인트
- 결말은 명확한 해피엔딩보다 **해석의 여지**를 남길 때 인상적
- 게임플레이 메커닉 자체가 **서사의 일부**가 되도록 설계
  - 예: 시간 되감기 = 후회 / 빛과 어둠 전환 = 이중성
- 텍스트 없이도 스토리가 전달되는 구조 고려

## 캐릭터 설계 포인트
- 말이 없거나 적은 주인공도 **행동 패턴으로 성격** 표현 가능
- NPC는 많지 않아도 되지만, 등장하면 **강하게 기억에 남아야** 함
- 빌런 없이 **환경·시스템·과거**가 적 역할을 할 수 있음

## 인디 게임다운 아이디어 방향
- "만약 ~라면?" 가정 하나를 끝까지 탐구
- 불편함·슬픔·상실을 게임 언어로 표현하기
- 플레이어가 게임을 끄고 나서도 생각하게 만드는 질문 하나 심기
